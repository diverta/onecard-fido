# OATH対応

最終更新日：2022/11/23

## 概要

[nRF52840アプリケーション](../../nRF52840_app)、[nRF5340アプリケーション](../../nRF5340_app)に、OATH機能（OATH-TOTP：ワンタイムパスワード生成機能など）を追加する対応です。

最終更新日現在、OATH機能は、[USB CCIDインターフェース](../../CCID/ccid_lib/README.md)上で動作します。
