# nRF5340アプリケーション

## 概要

現在開発計画中のMDBT53 Dongle（nRF5340）上で稼働するアプリケーションです。<br>
[nRF Connect SDK](https://www.nordicsemi.com/Software-and-tools/Software/nRF-Connect-SDK)を使用し、開発しています。
