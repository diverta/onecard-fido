# Android版 BLE近接認証アプリ

## 概要
<b>[BLE近接認証機能](../../../FIDO2Device/MDBT50Q_Dongle/BLEDAUTH.md)</b>において、[MDBT50Q Dongle](../../../FIDO2Device/MDBT50Q_Dongle/README.md)の基板上のボタンを押す代わりに、スマートフォンのボタンを押すことにより、FIDO認証を実行できるようにするアプリです。

#### 機能
* ペアリング実行
* BLE近接認証実行
