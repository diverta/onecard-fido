# BLE近接認証アプリ（仮称）

## 概要
[MDBT50Q Dongle](../../FIDO2Device/MDBT50Q_Dongle/README.md)の基板上のボタンを押す代わりに、スマートフォンのボタンを押すことにより、FIDO認証を実行できるようにするアプリです。

Android版と、iOS版を用意します。

## 機能
* ペアリング実行
* BLE認証実行
