# FIDO認証器管理ツール

## 概要
PC環境から、FIDO認証器の動作に必要な鍵・証明書の導入などを行うツールです。

macOS版と、Windows版を用意しました。

## 機能
* ペアリング実行
* 鍵・証明書削除
* 鍵・証明書インストール
* ヘルスチェック実行
* コマンドテスト機能
