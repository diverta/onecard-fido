# macOS版 FIDO認証器管理ツール

## 概要
PC環境から、[nRF52840版FIDO認証器](../../nRF5_SDK_v15.3.0)（以下単にFIDO認証器）の動作に必要な鍵・証明書の導入などを行うツールです。

### 機能
* ペアリング実行
* 鍵・証明書削除
* 鍵・証明書インストール
* PINコード設定
* CTAP2ヘルスチェック実行
* U2Fヘルスチェック実行
* コマンドテスト機能
* Flash ROM情報取得機能
