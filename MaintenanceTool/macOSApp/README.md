# macOS版 FIDO認証器管理ツール

## 概要
PC環境から、[FIDO認証器](../../nRF5_SDK_v15.3.0)の動作に必要な鍵・証明書の導入などを行うツールです。

### 機能
* ペアリング実行
* 鍵・証明書削除
* 鍵・証明書インストール
* PINコード設定
* CTAP2ヘルスチェック実行
* U2Fヘルスチェック実行
* コマンドテスト機能
* Flash ROM情報取得機能
* バージョン情報取得機能
* 自動認証パラメーター設定機能
* ログファイル格納ディレクトリー参照機能
